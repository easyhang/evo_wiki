# 沃伦·巴菲特

沃伦·巴菲特（Warren Buffett）是伯克希尔·哈撒韦（Berkshire Hathaway）的董事长兼 CEO，被称为“奥马哈先知”。师承本杰明·格雷厄姆，是价值投资最具影响力的实践者。

## 主要贡献

- 把保险**浮存金**转化为长期投资的低成本资金来源。
- 用通俗语言普及了 [[护城河]] 与 [[内在价值]] 等价值投资概念。
- 坚持“以合理价格买入伟大企业”，而非“以便宜价格买入平庸企业”。

## 关联
